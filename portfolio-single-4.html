Weather App is a real-time weather application built with Flutter, utilizing the OpenWeatherMap API to provide accurate weather data. Users can view local and global weather conditions with ease.

The app integrates location services to offer current location weather updates, providing personalized and up-to-date forecasts. With Bloc as the state management solution, the app delivers a smooth and efficient user experience, ensuring real-time data updates with minimal load times.
